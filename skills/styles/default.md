# 默认文风: 日系轻小说 + 轻松温馨向

## 核心结构
- 高比例对话 + 简短分段
- 动作、心理、对话分行处理
- 氛围描写适度长句铺陈，节奏轻快

## 叙述规则
- 叙述主观性: 代入主角视角，文章中存在主角主观的词语，避免完全客观
- 叙述情感性: 在叙述中体现角色情感，而非单纯客观描述
- 内心独白: 角色对事件进行思考或表达感想，正文中需体现
- 生动的动词: 以多样化的动词描写角色动作

## 对话规则
- 角色不是机器人，会说错话，有语气词和推断词
- 对话与叙述相辅相成，共同构建叙事
- 说话留白: 别把话说满，多停顿、岔开、含混带过，或只说半截

## 情感呈现
- 情绪表达以轻松、微甜、鲜活为主
- 内心独白带有浓烈主观意味
- 场景勾勒赋予意象意味
- 动作细化: 从主角所见所感出发，制造紧凑的亲密感

## 结构示例 (仅供参考，实际灵活多变)
```
[对话]
[以此展开的叙述 + 内心独白]
[对话]
[叙述 + 动作]
[对话]
[内心独白]
```

## 禁止
- 禁止空泛概括情绪 (如"他很高兴")
- 禁止比拟句和空洞修辞
- 禁止每轮都以环境描写起手、以心理描写收尾
- 禁止行文结构每次都一个样
